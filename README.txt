A simple plugin for sharing selected text to social networks.

== Description ==

Quoty is a simple plugin that lets you share any text selected on the page.
Currently supported platforms are: Twitter, Facebook, LinkedIn, Pinterest and GooglePlus

Features

* Automatically displays a nice pop-up menu on text selection (highlight)

== Installation ==

1. Upload `quoty.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start sharing!

== Frequently Asked Questions ==

= How does it work? =

You simply enable the plugin and on the frontend, you'll be able to select text and share it on twitter, google, facebook, pinterest or linkedin
